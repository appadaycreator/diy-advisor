# DIY難易度診断【無料】あなたのDIYスキルで何ができる？ - 技術仕様書

## 概要

**サービス名**: Diy Advisor
**バージョン**: 1.1.0
**更新日**: 2026-06-03
**URL**: https://appadaycreator.com/diy-advisor/

DIY経験・工具の有無・希望の作業から難易度と必要な道具を無料診断。初心者でもできるDIYを提案。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## v1.2.0 更新内容（2026-06-03）

### P1 実装（高優先度）

1. **利用者の声の DIY 固有化**
   - テンプレート的な利用者の声（プレゼント選び、ファッション、キャンプ用品）を削除
   - DIY 実施者の実践的な声に置き換え：
     - 初心者が壁紙張り替えで成功した体験
     - 床材交換の難易度判定で業者依頼を決めた利用者
     - DIY 経験がある判定で難易度低下の効果を実感した利用者
   - 理由：コンテンツ信頼性向上、DIY ユーザーへの共感形成

2. **診断結果の根拠説明（スコアリングロジック可視化）**
   - 難易度判定の根拠を結果セクションに明示
   - 経験値ボーナスの説明：「あなたの『そこそこある』という経験から -1 難易度を適用しました」
   - 計算式の透明化で信頼性向上
   - 理由：診断の根拠を明確にしてユーザー納得度向上

3. **工具のレンタルサービス案内追加**
   - 「工具を持っていない」ユーザーに購入以外の選択肢を提供
   - 難易度 ≤ 2 かつ工具なしの場合に自動表示
   - ホームセンター（カインズ・コーナン）のレンタルサービス案内：「丸鋸レンタル ¥500-1,000/日」
   - 理由：ユーザーの参入障壁を下げ、DIY への挑戦機会を増加

### P2 実装（中優先度）

1. **診断結果のコピー機能追加**
   - 「結果をコピー」ボタンで結果をテキスト形式でコピー
   - 形式：作業名・難易度・経験値・必要工具を整形
   - `navigator.clipboard.writeText()` で実装
   - コピー完了メッセージ表示（「✓ コピーしました！」）
   - 理由：メモアプリ・メールへの保存を簡単に、利便性向上

2. **FAQ の展開状態制御（初回は第1問のみ展開）**
   - 「電気・水道工事はできますか？」を初回 `open` 属性で展開
   - 重要な安全情報の視認性向上
   - モバイル時のスクロール量削減
   - 理由：安全面の重要な情報の認知率向上

3. **結果セクションでの視覚的な階層感改善**
   - 難易度判定の背景色・ボーダーを統一（緑=初心者 OK → 黄=要注意 → 赤=プロ推奨）
   - 工具リスト・手順番号の背景色も難易度色に合わせる
   - ユーザーの視認性向上で診断結果の理解が促進
   - 理由：カラーコーディングで直感的な難易度認知を支援

4. **モバイル対応の強化**
   - シェアボタンを `flex-col sm:flex-row` でレスポンシブ対応
   - 小画面では縦並び、中画面以上で横並び
   - タップターゲット 48px × 48px を確保
   - 理由：モバイル UX の改善、誤タップ防止
